# AI Existential Risk Probability Calculator

**AISC Team 19 · 2026**

An interactive probability calculator for estimating AI existential risk using a conditional probability chain across four branches: Intelligence, Alignment, Influence, and Environment.

## Quick Start

```bash
npm install
npm run dev
```

Open `http://localhost:5173` in your browser.

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) and import the repo
3. Vercel auto-detects Vite/React and deploys — no configuration needed

## Structure

- `src/App.jsx` — The full calculator (single component, all data inline)
- `index.html` — Entry point
- `src/main.jsx` — React mount
